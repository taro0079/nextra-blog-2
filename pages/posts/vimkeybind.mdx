# Vimの便利なKeybind

先日YoutubeをみていたらVimの使い方について説明している動画をみつけた。
そこで知ったkeybindがめちゃくちゃ便利だったのでメモ。

## 括弧の中だけを消したい
1. 変更したい箇所にカーソルを合わせる
2. 以下のコマンドを入力する。
```
ci)
```
これで括弧の中が消えて挿入モードになります。
youtubeのビデオの中でも言っていたが、Vimのキーバインドは覚えるモノじゃないらしい。
上記のコマンドの場合だと

Change Inside )

と連想できるとのこと。

Change Inside "

したければ
```
ci"
```
と入力。

また同じ考え方で
```
di)
```
とすれば括弧の中に含まれている文字を削除する。
